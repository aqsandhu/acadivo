## Role Description

The **Principal** is the head of a school or college. As a Principal, you have a complete overview of your institution's operations. You can:

- View school-wide statistics and analytics
- Post school announcements
- Send messages to teachers, students, and parents
- Monitor attendance and fee collection
- View reports on school performance
- Update school profile and settings
- Oversee all administrative activities

> **Note:** Principals cannot directly modify data managed by School Admins (like adding students or creating timetables) but can view everything.

## School Overview

### Viewing School Profile

1. Click **School Profile** from the sidebar or dashboard
2. View:
   - School name, code, and type
   - Address and contact information
   - Logo (if uploaded)
   - Subscription plan and expiry
   - Resource usage

### Updating School Profile

1. Click **Edit Profile**
2. Update fields:
   - Name, address, phone, email
   - Logo (upload new image)
   - School description
3. Click **Save Changes**

### Subscription Details

1. View current plan and features
2. See usage against limits:
   - Teachers: 15/25 used
   - Students: 340/500 used
3. Click **Upgrade** to request a higher plan (requires Super Admin approval)

## Managing Announcements

### Posting a School Announcement

1. Go to **Announcements > New Announcement**
2. Fill in the details:
   - **Title**: Short, clear headline
   - **Content**: Full message with details
   - **Priority**: NORMAL / HIGH / URGENT
   - **Target Audience**: ALL / TEACHERS / STUDENTS / PARENTS / ADMIN / Specific Class
   - **Attachments**: Optional (PDF, images)
   - **Pin**: Check to keep at the top of the noticeboard
   - **Expiry Date**: Optional auto-removal date
3. Click **Post Announcement**

### Announcement Priority Guide

| Priority | When to Use | Notification Sent |
|----------|------------|-----------------|
| **NORMAL** | Routine updates, events | In-app only |
| **HIGH** | Important deadlines, schedule changes | In-app + push notification |
| **URGENT** | Emergencies, closures | In-app + push + SMS |

### Managing Existing Announcements

1. Go to **Announcements > All**
2. View all announcements with filters
3. Actions:
   - **Edit**: Modify content or priority
   - **Pin/Unpin**: Keep at top or remove
   - **Delete**: Remove announcement
   - **View Stats**: See how many people viewed it

## Messaging

### Sending a Message

1. Go to **Messages > New Message**
2. Select recipient:
   - Search by name or role
   - Select individual or multiple recipients
3. Type your message
4. Attach files if needed (images, PDFs)
5. Click **Send**

### Viewing Conversations

1. Go to **Messages > Inbox**
2. Conversations are listed with:
   - Sender name and avatar
   - Last message preview
   - Timestamp
   - Unread count badge
3. Click any conversation to open the chat

### Message Types

| Type | Description | Max Size |
|------|-------------|----------|
| **Text** | Plain text messages | 2000 chars |
| **Image** | Photos, screenshots | 10 MB |
| **File** | PDFs, documents | 20 MB |
| **Voice** | Audio messages | 5 MB |

## Viewing Reports

### Available Reports

1. Go to **Reports > School Reports**
2. Select report type:

| Report | Description |
|--------|-------------|
| **Enrollment Summary** | Student count by class and gender |
| **Attendance Summary** | Monthly attendance trends |
| **Fee Collection Report** | Collection vs. target by month |
| **Teacher Performance** | Classes taught, attendance marked |
| **Academic Performance** | Class-wise pass percentages |
| **Behavior Report** | Discipline incidents summary |

### Generating a Report

1. Select report type
2. Choose parameters (date range, class, etc.)
3. Click **Generate**
4. View on screen or download as PDF

### Viewing Fee Status

   - Total collected

   - Collection percentage
   - Overdue amounts

1. Go to **Fee > Trends**

## Attendance Monitoring

### Daily Attendance Overview

1. Go to **Attendance > Daily Overview**
2. Select date
3. View:
   - Overall attendance percentage
   - Present, absent, late, leave counts
   - Class-wise breakdown
   - Students with low attendance (alert list)

### Low Attendance Alerts

Students with attendance below **75%** are flagged:

1. Go to **Attendance > Alerts**
2. View list of at-risk students
3. Click **Notify Parents** to send automatic alert
4. Or click **View Details** to see attendance history

### Monthly Attendance Report

1. Go to **Attendance > Monthly Report**
2. Select month and year
3. Generate report with:
   - Class-wise attendance percentages
   - Teacher-wise attendance marking
   - Trends compared to previous month

## Managing Staff and Students

### Viewing Teachers

1. Go to **Staff > Teachers**
2. View all teachers with:
   - Name and contact
   - Classes assigned
   - Subjects taught
   - Attendance marking status
3. Click a teacher to view their profile and timetable

### Viewing Students

1. Go to **Students > All Students**
2. Filter by:
   - Class and section
   - Status (Active, Inactive, Graduated)
   - Attendance percentage
3. Click a student to view their complete profile

### Viewing Parents

1. Go to **Parents > All Parents**
2. View parents with:
   - Name and contact
   - Children enrolled
   - Fee payment status
3. Click a parent to message them or view their children

## FAQ

**Q: Can a Principal edit student records?**
A: No, Principals have read-only access to student data. Only School Admins can add, edit, or remove students.

**Q: How do I know if a teacher is marking attendance regularly?**
A: Go to Attendance > Teacher Activity to see which teachers have marked attendance and which haven't.

**Q: Can I send an announcement to only one class?**
A: Yes! When creating an announcement, select Target Audience: CLASS, then choose the specific class and section.

**Q: How do I export attendance data for government reporting?**
A: Go to Attendance > Reports, generate the monthly report, and click Download as Excel. The format meets government reporting standards.

**Q: What happens when a parent's fee is overdue?**
A: The system sends automatic reminders after 5 days, 15 days, and 30 days. After 30 days, the student may be flagged for administrative action.

**Q: Can I access Acadivo on my phone?**
A: Yes! Use the Acadivo mobile app (available on Play Store and App Store) or access the web app from your mobile browser.

## Tips and Tricks

1. **Weekly Summary**: Every Monday, the system sends an automatic weekly summary email with attendance and fee highlights

2. **Quick Stats**: Hover over any number on the dashboard to see the previous period's comparison

3. **Announcement Scheduling**: Write announcements in advance and schedule them to post automatically

4. **Bulk Messaging**: Hold Ctrl (or Cmd on Mac) while selecting recipients to message multiple people at once

5. **Export Anywhere**: Almost every table and report has an export button for Excel or PDF

6. **Print-Friendly**: Press Ctrl+P on any report page for a clean print layout

7. **Urdu Support**: Switch language to Urdu from the top-right menu for Urdu interface
