## Role Description

**Teachers** are the backbone of the Acadivo platform. As a Teacher, you can:

- Mark daily attendance for your classes
- Create and assign homework
- Enter marks for exams and assessments
- Answer student and parent questions in the Q&A forum
- Generate progress and attendance reports
- View your teaching timetable
- Communicate with students, parents, and colleagues

> **Note:** You only see data for classes and subjects assigned to you by the School Admin.

## Attendance Marking

### Marking Daily Attendance

1. Go to **Attendance > Mark Attendance**
2. Select:
   - Class
   - Section
   - Date (default: today)
3. The student list appears
4. For each student, select status:
   - **Present** (green)
   - **Absent** (red)
   - **Late** (yellow)
   - **Leave** (blue)
   - **Half Day** (orange)
5. Add remarks for absent/late students (optional)
6. Click **Save Attendance**

### Bulk Actions

- **Mark All Present**: Click the button at the top, then adjust individual students
- **Previous Day's Pattern**: Copy yesterday's attendance as a starting point

### Viewing Attendance Records

1. Go to **Attendance > History**
2. Filter by class, section, date range
3. View individual student attendance percentages
4. Export as Excel or PDF

### Attendance Summary

1. Go to **Attendance > Summary**
2. Select month
3. View class-wise attendance percentages
4. Identify students with low attendance for follow-up

## Homework Management

### Creating Homework

1. Go to **Homework > Create**
2. Fill in:
   - **Title**: Brief description (e.g., "Chapter 5 Exercises")
   - **Description**: Full instructions
   - **Class and Section**
   - **Subject**
   - **Due Date and Time**
   - **Maximum Marks** (optional)
   - **Attachments**: Upload files (worksheets, reference material)
3. Click **Assign**
4. Students receive notification immediately

### Viewing Submissions

1. Go to **Homework > Submissions**
2. Filter by homework assignment
3. View list of students:
   - Submitted (green)
   - Pending (yellow)
   - Late (red)
4. Click any student to view their submission

### Grading Submissions

1. Open a student's submission
2. Review uploaded files and text
3. Enter marks (out of maximum)
4. Add feedback/comments
5. Click **Save Grade**
6. Student receives notification with grade

### Homework Status Guide

| Status | Meaning |
|--------|---------|
| **Pending** | Student hasn't submitted yet |
| **Submitted** | Submitted before or on due date |
| **Late** | Submitted after due date |
| **Graded** | Teacher has reviewed and graded |

## Marks Entry

### Entering Marks

1. Go to **Marks > Enter Marks**
2. Select:
   - Class
   - Section
   - Subject
   - Exam type (Quiz, Midterm, Final, Assignment, Project)
3. Student list appears with empty marks fields
4. For each student:
   - Enter total marks (e.g., 100)
   - Enter obtained marks
   - Grade auto-calculates based on school grading scheme
   - Add remarks (optional)
5. Click **Save Marks**

### Bulk Entry

1. Enable **Bulk Entry Mode**
2. Enter marks in a spreadsheet-like interface
3. Tab between cells for faster entry
4. Click **Save All**

### Viewing Marks History

1. Go to **Marks > History**
2. Filter by class, subject, exam type
3. View all marks entries
4. Click **Edit** to correct any mistakes

## Q&A Forum

### Viewing Questions

1. Go to **Q&A > My Questions**
2. View questions from students and parents
3. Filter by:
   - Answered / Unanswered
   - Subject
   - Date range

### Answering a Question

1. Click on a question
2. Type your answer in the text box
3. Attach files if needed (diagrams, examples)
4. Click **Post Answer**
5. The asker receives a notification

### Asking a Question

1. Go to **Q&A > Ask**
2. Select subject category
3. Type your question
4. Add attachments (optional)
5. Click **Post Question**

### Best Practices

- Answer within 24-48 hours
- Be clear and concise
- Include examples where helpful
- If a question is off-topic, politely redirect

## Report Generation

### Types of Reports

1. Go to **Reports > Generate**
2. Select report type:

| Report | Contents |
|--------|----------|
| **Progress Report** | Marks, attendance, overall grade |
| **Attendance Report** | Detailed attendance with percentages |
| **Behavior Report** | Conduct and discipline notes |
| **Comprehensive Report** | All of the above combined |

### Generating a Report

1. Select report type
2. Select student(s)
   - Single student
   - Entire class
   - Entire section
3. Select parameters:
   - Date range / Term
   - Include remarks (yes/no)
4. Click **Generate**
5. Preview the report
6. Download as PDF or print

### Report Request from Parents

When a parent requests a report:
1. Go to **Reports > Requests**
2. View pending requests
3. Click **Generate** next to the request
4. The report is generated and parent is notified

## Messaging

### Sending Messages

1. Go to **Messages > New**
2. Search for recipient by name or select from class list
3. Type your message
4. Attach files if needed
5. Click **Send**

### Receiving Messages

- Notifications appear in the top bar
- Click the bell icon to see recent messages
- Go to **Messages > Inbox** to see all conversations

### Group Messaging

1. Go to **Messages > New Group**
2. Select multiple recipients
3. Type group message
4. Click **Send**

## FAQ

**Q: Can I mark attendance for a previous date?**
A: Yes, but only for dates within the current academic week. Go to Attendance > Mark Attendance and change the date.

**Q: What if a student joins my class mid-year?**
A: The School Admin will add the student to your class. They will appear in your attendance and marks lists automatically.

**Q: Can I edit marks after saving?**
A: Yes, go to Marks > History, find the entry, and click Edit. Changes are tracked in the audit log.

**Q: How do I handle a parent who disagrees with a grade?**
A: You can update the marks if there was a genuine error. For disputes, involve the Principal through the messaging system.

**Q: Can I create homework for multiple sections at once?**
A: Yes! When creating homework, you can select multiple sections of the same class.

**Q: What happens if I don't mark attendance?**
A: The system will remind you. After 3 days, a notification is sent to the Principal.

## Tips and Tricks

1. **Quick Attendance**: From the dashboard, click any class card to jump directly to attendance marking

2. **Homework Templates**: Save frequently used homework as templates for quick assignment

3. **Marks Preview**: Before saving, preview the grade distribution to catch any data entry errors

4. **Voice Messages**: In messaging, you can send voice notes for quick communication

5. **Offline Mode**: The mobile app works offline. Mark attendance and it syncs when you're back online

6. **Urdu Interface**: Switch to Urdu from settings if you prefer working in Urdu

7. **Keyboard Shortcuts**:
   - `Tab`: Move to next student in marks entry
   - `Ctrl+S`: Save current attendance
   - `Escape`: Cancel current operation
